# Instructions

## 1. Clone Source Repository
$ git clone https://github.org/orionblastar/K666
$ cd K666

## 2. Create the environment
$ . ./k666-env

This starts the server, it should start up ready.

## 3. Visit the site.
Go to http://localhost:8000/

We have begun.
